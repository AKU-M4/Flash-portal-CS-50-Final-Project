Flash Portal

Welcome to Flash Revival Games

This project brings back the joy of classic Flash games to a modern audience. Built as part of my CS50 final project, this website celebrates nostalgia while providing a fresh, engaging experience.

Project Goals

• Play classic Flash games directly in your browser

• User accounts for saving progress and achievements

• Earn coins by completing challenges and unlock new games (Coming soon)

Technology Stack

HTML CSS Bootstrap JavaScript Python Flask SQLite Git

Challenges and Learning

Building this project was an enriching experience. Integrating a transaction system and managing user progression were particularly challenging but allowed me to deepen my understanding in backend logic and database management.
